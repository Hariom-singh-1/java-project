# 🎬 Movie Ticket Booking System

A modern and user-friendly Java-based Movie Ticket Booking System designed to simplify the process of browsing movies, selecting seats, and booking tickets efficiently.

---

## ✨ Overview

The Movie Ticket Booking System is a console-based application developed using Java. It allows users to explore available movies, choose preferred show timings, select seats, and book tickets seamlessly. The project demonstrates core Java programming concepts and provides a practical implementation of a real-world booking system.

---

## 🎯 Features

🎥 View available movies and show timings

🎫 Book movie tickets easily

💺 Select preferred seats

👤 Customer information management

📋 Display booking details and ticket summary

🔍 Search and browse movies

❌ Cancel bookings when required

⚡ Fast and simple user interaction

## 🚀 How It Works

1. Start the application.
2. Browse available movies.
3. Select a movie and show timing.
4. Choose available seats.
5. Enter customer details.
6. Confirm booking.
7. View booking summary and ticket details.
